# Sometime

Simple and compact timepicker.

- See [demo](http://cognitom.github.io/sometime-riot/).

![demo](http://cognitom.github.io/sometime-riot/demo.gif)

## Install

```bash
$ npm install --save sometime-riot
```

## Usage

```html
<time-picker value={ time } onchange={ change } />
```

## Try or contribute

Clone this repo.

```bash
$ npm install
$ npm start
```
